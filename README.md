# BOZ213v01a01
```
Bu oyun python ile yazılmış bilgisayara karşı oynanan bir tic-tac-toe, bilinen adıyla XOX oyunudur.
Oyuna bilgisayar başlar ve ilk hamlesini her zaman ortadaki kareye oynar.
```

## Kurallar
- Kullanıcı - ```O``` , Bilgisayar - ```X```
- Kullanıcı 1-9 arsında bir sayı seçmek zorunda
- Daha önce seçilmiş karelere tekrar oynanamaz

## Kazanma
Oyun "Bilgisayar kazandı!", "Kazandın!" ve "Berabere!" olmak üzere 3 farklı geri bildirimde bulunmaktadır.
- Alt alta 3 tane aynı işaretin gelmesi
- Yanyana 3 tane aynı işaretin gelmesi
- Çarpraz şekilde 3 tane aynı işaretin gelmesi

## Kullanılan Teknolojiler
- Python 3.x
- ```random``` (bilgisayarın rastgele bi kare seçmesi için)

## Kullanmak İçin
### Ön Koşullar
Bu projeyi çalıştırmak için bilgisayarınızda yalnızca ```Python 3``` kurulu olmalıdır.
### Kurulum ve Çalıştırma
- Proje dosyasını yerel olarak indirin/oluşturun.
- Komut satırını (Terminal/CMD) açın ve dosyanın bulunduğu dizine gidin.
- Aşağıdaki komutu kullanarak oyunu başlatın:
  ```
  bash
  python Arasınav_Projesi.py
  ```
